<p class=MsoNormal><b style='mso-bidi-font-weight:normal'>Introduction<o:p></o:p></b></p>

<p class=MsoNormal>Please have a look at <a
href="https://github.com/NKI-Pathology/QDNAseqFlow/blob/master/QDNAseqFlow-Abstract.pdf"
title=QDNAseqFlow-Abstract.pdf>QDNAseqFlow-Abstract.pdf</a> and <a
href="https://github.com/NKI-Pathology/QDNAseqFlow/blob/master/QDNAseqFlow_poster_ISMB.pdf"
title="QDNAseqFlow_poster_ISMB.pdf">QDNAseqFlow_poster_ISMB.pdf</a> for an
introduction.</p>

<p class=MsoNormal><b style='mso-bidi-font-weight:normal'>Usage<o:p></o:p></b></p>

<p class=MsoNormal style='margin-left:18.0pt'>This workflow consists of 3 main
R-programs:</p>

<p class=MsoListParagraphCxSpFirst style='margin-left:54.0pt;mso-add-space:
auto;text-indent:-18.0pt;mso-list:l1 level1 lfo1'><![if !supportLists]><span
style='mso-bidi-font-family:Calibri;mso-bidi-theme-font:minor-latin'><span
style='mso-list:Ignore'>1.<span style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</span></span></span><![endif]>QDNAseq_BAM2CopyNumbers.R</p>

<p class=MsoListParagraphCxSpMiddle style='margin-left:54.0pt;mso-add-space:
auto;text-indent:-18.0pt;mso-list:l1 level1 lfo1'><![if !supportLists]><span
style='mso-bidi-font-family:Calibri;mso-bidi-theme-font:minor-latin'><span
style='mso-list:Ignore'>2.<span style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</span></span></span><![endif]>QDNAseq_CopyNumbers2PlotsCallsSegmentFileInclDewave.R</p>

<p class=MsoListParagraphCxSpLast style='margin-left:54.0pt;mso-add-space:auto;
text-indent:-18.0pt;mso-list:l1 level1 lfo1'><![if !supportLists]><span
style='mso-bidi-font-family:Calibri;mso-bidi-theme-font:minor-latin'><span
style='mso-list:Ignore'>3.<span style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</span></span></span><![endif]><span class=SpellE>QDNAseq_FrequencyCGHregionsCGHtest.R</span></p>

<p class=MsoNormal style='margin-left:18.0pt'>All of them can be run
interactively, which means that simple GUIs pop-up at runtime that allow the
user to select input files and parameters. The first two scripts can also be run
with no GUI with command line options.</p>

<p class=MsoNormal>There are basically 3 equivalent ways to run the scripts:</p>

<p class=MsoListParagraphCxSpFirst style='text-indent:-18.0pt;mso-list:l0 level1 lfo4'><![if !supportLists]><span
style='mso-bidi-font-family:Calibri;mso-bidi-theme-font:minor-latin'><span
style='mso-list:Ignore'>1.<span style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</span></span></span><![endif]>From <span class=SpellE>Rstudio</span>: open
script file and execute with CTRL-ALT-R</p>

<p class=MsoListParagraphCxSpMiddle><o:p>&nbsp;</o:p></p>

<p class=MsoListParagraphCxSpLast style='text-indent:-18.0pt;mso-list:l0 level1 lfo4'><![if !supportLists]><span
style='mso-bidi-font-family:Calibri;mso-bidi-theme-font:minor-latin'><span
style='mso-list:Ignore'>2.<span style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</span></span></span><![endif]>From the command line window:</p>

<p class=MsoNormal style='margin-left:36.0pt'>/path/to/your/r-installation/<span
class=SpellE>Rscript</span> QDNAseq_BAM2CopyNumbers.R –interactive</p>

<p class=MsoNormal style='margin-left:36.0pt'>or with no options or with –help to
get an overview of available command line options.</p>

<p class=MsoNormal style='margin-left:36.0pt'>Likewise for the second script:</p>

<p class=MsoNormal style='margin-left:36.0pt'>/path/to/your/r-installation/<span
class=SpellE>Rscript</span> QDNAseq_BAM2CopyNumbers.R –interactive</p>

<p class=MsoNormal style='margin-left:36.0pt'>For the third script only:</p>

<p class=MsoNormal style='margin-left:36.0pt'>/path/to/your/r-installation/<span
class=SpellE>Rscript</span> <span class=SpellE>QDNAseq_FrequencyCGHregionsCGHtest.R</span></p>

<p class=MsoNormal style='margin-left:36.0pt'>Note: Type Rscript.exe on
Windows.</p>

<p class=MsoListParagraphCxSpFirst style='text-indent:-18.0pt;mso-list:l0 level1 lfo4'><![if !supportLists]><span
style='mso-bidi-font-family:Calibri;mso-bidi-theme-font:minor-latin'><span
style='mso-list:Ignore'>3.<span style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</span></span></span><![endif]>Via a Batch file (.bat, on Windows only)</p>

<p class=MsoListParagraphCxSpMiddle>Edit the bat file with a text editor to
adjust the path to your r-installation. Please note that the path must not
contain white spaces, which means that R <span class=SpellE>can not</span> be
installed at the default location on “program files”, because otherwise no
packages can be installed or compiled. This is ridiculous, please ask the R-developers.
</p>

<p class=MsoListParagraphCxSpMiddle><o:p>&nbsp;</o:p></p>

<p class=MsoListParagraphCxSpMiddle style='margin-left:0cm;mso-add-space:auto'><b
style='mso-bidi-font-weight:normal'>Required R-version<o:p></o:p></b></p>

<p class=MsoListParagraphCxSpMiddle>We have made most tests with R 3.4.1. It
might work with older R versions but the risk is that some required libraries
might not be available.</p>

<p class=MsoListParagraphCxSpLast style='margin-left:0cm;mso-add-space:auto'><o:p>&nbsp;</o:p></p>

<p class=MsoNormal><span class=SpellE><b style='mso-bidi-font-weight:normal'>Dewaving</b></span><b
style='mso-bidi-font-weight:normal'><o:p></o:p></b></p>

<p class=MsoNormal style='margin-left:36.0pt'>The <span class=SpellE>dewaving</span>
step is optional. Although it relies on published code (<span class=SpellE>NoWave</span>
program), right now we only have a proprietary reference dataset available that
is based on NGS data. We would need normal samples, ideally for every tumor
type and are currently looking for such data (if you can help us, please let us
know). For now, running the <span class=SpellE>dewave</span> step requires a
key, due to the proprietary reference data.<o:p></o:p></p>

<p class=MsoNormal><b style='mso-bidi-font-weight:normal'><o:p>&nbsp;</o:p></b></p>

<p class=MsoNormal><b style='mso-bidi-font-weight:normal'>Odd things, to be
improved<o:p></o:p></b></p>

<p class=MsoNormal style='margin-left:18.0pt'>Run and quit: Right now, when run
from command line or from the bat file, the program just runs till the end
closes the command window. Don’t be surprised. I will make the programs more
communicative.<b style='mso-bidi-font-weight:normal'><o:p></o:p></b></p>

<p class=MsoListParagraph style='margin-left:18.0pt;mso-add-space:auto'>Logging:
For reproducibility and quality control, extensive logging of versions of used
packages and progress is envisioned.</p>
